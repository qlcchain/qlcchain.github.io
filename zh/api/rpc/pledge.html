# pledge_getPledgeData

根据抵押参数返回抵押的合约数据

  • Parameters:

    • pledgeParams : 抵押参数
      • beneficial ：受益人地址
      • pledgeAddress : 抵押地址
      • amount ：抵押数量
      • pType ：抵押类型
      • nEP5TxId : nep5 的抵押交易 id

  • Returns:

    • string : 抵押的合约数据

  • Example:

{
	"jsonrpc": "2.0",
	"id": 1,
	"method": "pledge_getPledgeData",
	"params": [
		{
