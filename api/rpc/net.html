# net_peersCount

Return peers count

  • Parameters: none

  • Returns:

    • peersCount
    • all : all nodes in the network
      • connect : how many nodes are connected
      • online : online nodes in the network

  • Example:

{
  "jsonrpc": "2.0",
  "id":2,
  "method":"net_peersCount"
}
{
  "jsonrpc": "2.0",
  "id": 18,
  "result": {
    "all": 3,
    "connect": 2,
    "online": 3
  }
}
{
  "jsonrpc": "2.0",
  "id":2,
  "method":"net_peersCount"
}

# net_syncing

Return sync status

  • Parameters: null
  • Returns:
    • sync status :
      • true ： syncing now
      • false ： not syncing
  • Example:
{
  "jsonrpc": "2.0",
  "id":3,
  "method":"net_syncing"
}

{
	"jsonrpc": "2.0",
	"id": 3,
	"result": false
}

{
  "jsonrpc": "2.0",
  "id":3,
  "method":"net_syncing"
}

# net_getBandwidthStats

Return bandwidth metrics

  • Parameters: null
  • Returns:
    • bandwidth stats :
      • TotalIn TotalOut ： The TotalIn and TotalOut fields record cumulative bytes sent / received.
      • RateIn RateOut ： The RateIn and RateOut fields record bytes sent / received per second.
  • Example:
{
	"jsonrpc": "2.0",
	"id":18,
	"method":"net_getBandwidthStats"
}
{
  "jsonrpc": "2.0",
  "id": 18,
  "result": {
    "TotalIn": 598546022,
    "TotalOut": 3524381031,
    "RateIn": 7498.377919815488,
    "RateOut": 28352.544366668382
  }
}
{
	"jsonrpc": "2.0",
	"id":18,
  	"method":"net_getBandwidthStats"
}
